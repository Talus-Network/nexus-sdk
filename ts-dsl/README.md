# @nexus/dag-dsl

Type-safe TypeScript DSL for constructing Nexus DAGs — a direct mirror
of the Rust [`nexus-dag-dsl`](../dag-dsl/) crate. A DAG authored in
either language produces byte-identical wire JSON, pinned by
cross-language round-trip tests in CI.

## Surface

- **Relaxed** (`DagBuilder`) — string-keyed ports, validation at
  `.build()`. Works with any tool without a descriptor.
- **Strict** (`TypedDagBuilder`) — `InPort<T>` / `OutPort<V, T>` with
  phantom generics; `connect_*` signatures enforce edge-kind arity
  (e.g. `connectForEach` requires `OutPort<V, T[]>` → `InPort<T>`).
- **Flat** — one method per edge kind; 1:1 with the wire format.
- **Scoped** — `forEach` / `doWhile` closures; scope-local vertex
  names auto-prefixed with `.`.

## Quick start — relaxed

```ts
import { DagBuilder } from "@nexus/dag-dsl";

const dag = new DagBuilder();
const a = dag.offchain("a", "xyz.taluslabs.math.i64.add@1");
const b = dag.offchain("b", "xyz.taluslabs.math.i64.add@1");
dag
  .entryPort(a, "a")
  .inlineDefault(a, "b", 1)
  .edge(a.out("ok", "result"), b.inp("a"))
  .inlineDefault(b, "b", 2)
  .output(b.out("ok", "result"));

const result = dag.build();
if (!result.ok) throw new Error("invalid DAG");
const wire = result.dag; // can be JSON.stringify'd for consumers
```

## Quick start — strict (descriptor + typed connect)

```ts
import { defineTool, TypedDagBuilder } from "@nexus/dag-dsl";

const AddTool = defineTool({
  fqn: "xyz.taluslabs.math.i64.add@1",
  inputs: { a: 0 as unknown as number, b: 0 as unknown as number },
  outputs: { ok: { result: 0 as unknown as number } },
});

const dag = new TypedDagBuilder();
const a = dag.add("a", AddTool);
const b = dag.add("b", AddTool);
dag.entryPort(a, "a");
dag.inlineDefault(a.inp.b, 1);
dag.inlineDefault(b.inp.b, 2);
dag.connect(a.out.ok.result, b.inp.a); // compile-time type check
dag.output(b.out.ok.result);
```

Tools whose descriptors are auto-generated by `nexus-dsl-codegen`
produce the same shape and are drop-in usable.

## Scripts

- `npm run typecheck` — `tsc --noEmit`.
- `npm run build` — emit `dist/` (ESM).
- `npm test` — vitest; includes the cross-language round-trip against
  the Rust SDK's committed fixtures (`sdk/src/dag/_dags/`).
- `npm run lint` — prettier check.

## Wire-format contract

The canonical wire contract is the Rust `nexus_sdk::types::Dag` struct
(see [`sdk/src/types/json_dag.rs`](../sdk/src/types/json_dag.rs)). This
package's `src/types.ts` mirrors that struct; any divergence is caught
by the round-trip test in `tests/roundtrip.test.ts`.
